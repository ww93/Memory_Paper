UMEM: Unified Memory Extraction and Management Framework for Generalizable Memory 关心的不是“记忆库怎么检索”，而是“从经验里抽取什么记忆、又如何管理这些记忆”必须被联合优化。论文把记忆优化器本身作为可训练策略，用 Semantic Neighborhood Modeling 避免抽到只对单个样本有用的噪声。

1. 核心判断

UMem 把长期记忆看作 agent 的“可演化参数”：冻结 executor 做任务，memory optimizer 从执行轨迹中抽取 insight 并更新 memory bank。它的重点是让记忆从实例经验中泛化，而不是把每条轨迹里的细节都塞进库里。

一句话概括：UMem 不是只训练“怎么增删改查记忆”，而是同时训练“什么才值得被抽成记忆”和“这些记忆如何并入库”。这让 extraction 和 management 不再错位。

适合的场景
