📚 Memory Paper

论文解读与交互式演示集合

📖 如何使用

  • 每篇论文展示在卡片中，点击「查看」按钮访问详细解读
  • 所有论文的 HTML 页面均可通过 GitHub Pages 公网访问
  • 论文按独立目录组织：papers/[paper-id]/index.html
  • 访问链接格式：https://[username].github.io/Memory_Paper/papers/[paper-id]/

➕ 添加新论文

  • 在 papers/ 目录下创建新文件夹：papers/[new-paper-id]/
  • 将论文 HTML 文件放入该目录并命名为 index.html
  • 更新本页面的 papers.json 数据（或重新加载自动检测）
  • Push 到 GitHub 后，新论文自动通过 GitHub Pages 发布

🔗 GitHub Pages 配置

  • 无需额外配置。GitHub Pages 已默认启用，根分支（main/master）即为发布源
  • 确保仓库名为 Memory_Paper（公开仓库）
  • 所有 HTML 文件自动可通过 https://[username].github.io/Memory_Paper/[path] 访问
  • 为了最佳效果，建议设置 GitHub Pages 使用 main 分支根目录
