MSA

Memory Sparse Attention

RAG 长上下文 记忆系统 注意力机制

📋 论文简介

Memory Sparse Attention (MSA) 是一种端到端潜在记忆扩展框架。 核心创新不是构建外部检索器，而是将文档离线编码成模型内部的压缩 KV 缓存， 通过 Top-k 稀疏注意力、文档级 RoPE 和 Memory Parallel 实现记忆容量与推理上下文的完全解耦。

该方案支持 100M 词元级记忆，同时单次推理上下文被 Top-k 和压缩率约束， 使得模型能在有限硬件上存得下、查得快、算得动。

🎯 核心创新

✓ 端到端潜在记忆

文档离线编码为压缩 K/V，可直接进入模型注意力计算

✓ Top-k 稀疏路由

Router-Q 匹配文档 Router-K，选择最相关的少量记忆

✓ 文档级 RoPE

每个文档独立位置编码，解耦记忆库规模和位置外推风险

✓ Memory Interleave

多轮迭代检索，支持复杂多跳推理和动态证据链

📊 主要成果

• 平均分数：3.760（基于 Qwen3-4B），超越同基座 RAG 系统 16%+
• 长记忆稳定性：100M 词元下仅下降 8.8%
• 上下文稳定性：1M 词元 NIAH 准确率 94.84%
• 消融实验：路由预训练下降 31.3%，Memory Interleave 下降 5.3%
